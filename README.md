# Getting started
Here is how to get started:
1. Download or clone the repo `git clone https://github.com/kyunwang/hangman-chat.git`
2. Run `npm install` or `yarn`
3. Do a `npm start`
4. Go to **localhost:3100** and you are ready to go.
